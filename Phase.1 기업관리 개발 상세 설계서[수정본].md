### 4. 기업 테이블

#### 4-1. 컬럼 정의

| 순서 | 컬럼       | 너비      | 설명                                    |
| ---- | ---------- | --------- | --------------------------------------- |
| 0    | ☐ 체크박스 | 40px 고정 | 행 선택. 헤더 체크박스 = 전체 선택/해제 |
| 1    | 기업명     | flex 1.5  | 클릭 시 사이드 드로어 열림              |
| 2    | 소재지     | flex 1    | 주소 전체 또는 시/구까지 truncate       |
| 3    | 대표자     | 100px     | 대표자 이름                             |
| 4    | 협약상태   | 90px      | 체결 / 미체결 Badge                     |
| 5    | 참여과정   | flex 1.2  | 과정 유형 태그 (훈련/지원/세미나)       |

#### 4-2. 셀 표시 규칙

- **기업명:**
  - 폰트: 14px, semibold
  - 하단에 사업자번호를 작은 회색 텍스트로 표시 (선택)
- **협약서 여부:**
  - 체결 → 녹색 뱃지 `체결`
  - 미체결 → 회색 뱃지 `미체결`
- **참여 과정:**
  - 태그(칩) 형태로 표시
  - 훈련비 → 녹색 태그 `훈련`
  - 지원비 → 파란 태그 `지원`
  - 세미나 → 노란 태그 `세미나`
  - 복수 참여 시 태그 나란히 표시: `훈련` `지원` `세미나`

#### 4-3. 행 인터랙션

- **Hover:**
  - 행 전체 배경색 변경 (light gray)
  - 각 태그칩 hover → 세부 교육 과정 툴팁 표시
  - cursor: pointer
- **Click:**
  - 행 클릭 → 해당 기업의 사이드 드로어 오픈
  - 클릭된 행에 좌측 보더 강조 (active 상태)
  - 체크박스 클릭 → 하단 플로팅 버튼 오픈 (섹션 11 참조)
- **정렬:**
  - 컬럼 헤더 클릭 시 오름차순/내림차순 토글
  - 기본 정렬: 기업명 가나다순
- **페이지네이션:**
  - 하단에 페이지네이션 or 무한 스크롤
  - 한 페이지 기본 20개

### 5. 사이드 드로어 — 기업 상세

#### 5-1. 드로어 기본 스펙

- **위치:** 우측에서 슬라이드 인
- **너비:** 480px ~ 560px (반응형 시 화면의 40%)
- **오버레이:** 반투명 배경 (클릭 시 드로어 닫힘)
- **애니메이션:** 우측에서 좌측으로 300ms ease-out

#### 5-2. 드로어 헤더

- **기업명:** 20px bold, 좌측 정렬
  - 더블 클릭시 기업명 인라인 편집 모드 진입
- **우측 아이콘:**
  - `[>]` 닫기 아이콘 → 드로어 닫기

#### 5-3. 기업 정보 섹션

- **섹션 제목:** "기업 상세 정보"
  - 우측에 `[편집]` 연필 아이콘
- **표시 필드 (읽기 모드):**
  - 2컬럼 그리드 레이아웃
    ```
    대표명      김도현      사업자 번호 : 123-45-67890
    소재지    광주 북구 첨단과기로 123
    ---
    담당자    박소영
    연락처      010-1234-5678     이메일    hr@hanbit-tech.co.kr
    협약서      ✅ 체결            체결일    2026-01-15
    ```
  - Label: 12px, 회색
  - Value: 14px, 검정, medium weight
- **편집 모드 진입 시:**
  - 각 Value가 input/select 필드로 전환
  - 협약서 여부 → 토글 스위치 또는 드롭다운
  - 체결일 → 날짜 선택기 (date picker)
  - 하단에 `[저장]` `[취소]` 버튼 노출
  - `Enter` → 저장, `Esc` → 취소

#### 5-4. 교육 과정 섹션

- **섹션 제목:** "교육 과정 현황"
- **과정 구분별 그룹핑:**
  - 과정 타입 헤더: 아이콘 + 텍스트
    - `🟢 훈련비과정`
    - `🔵 지원비과정`
    - `🟡 공유개방 세미나`
  - 각 타입 아래 구분선 (divider, 점선)

- **세부 과정 행 (접힘 상태):**

  ```
  ▸ 스마트팩토리 실무     2026.03.15~03.17 · 24h · 참여 3명
  ```

  - 좌측: 펼침/접힘 화살표 (`▸` / `▾`)
  - 과정명: 14px, medium
  - 우측 메타 정보: 12px, 회색
    - 진행일 (시작일~종료일)
    - 교육 시간
    - 참여 인원 수
  - Hover 시: 행 배경색 변경 + `[삭제]` 아이콘 우측 끝에 표시

- **세부 과정 행 (펼침 상태):**

  ```
  ▾ 스마트팩토리 실무     2026.03.15~03.17 · 24h
    ┌─────────────────────────────────────────┐
    │ 참여자         수료 상태                  │
    │ 박진우 ⓘ       🟢 수료                  │
    │ 김민지 ⓘ       ⚪ 미수료                │
    │ 이태호 ⓘ       🟢 수료                  │
    │                                         │
    │ [+ 참여자 추가]                          │
    └─────────────────────────────────────────┘
  ```

  - 참여자 리스트: 테이블 또는 리스트 형태
  - 각 참여자 행 구성:
    - **이름:** 14px, 링크 스타일 (hover 시 밑줄)
    - **ⓘ 아이콘:** 이름 옆, hover 시 Popover 카드 표시 (섹션 5-5 참조)
    - **수료 상태:**
      - `🟢 수료` — 녹색 텍스트
      - `⚪ 미수료` — 회색 텍스트
    - **Hover 시:** 행 우측에 `[삭제]` 아이콘 표시
  - **`[+ 참여자 추가]` 버튼:**
    - 클릭 시 → 인라인 입력 행 추가 or 참여자 선택 드롭다운
    - 기존 등록된 참여자 중 선택 or 신규 참여자 등록

#### 5-5. 참여자 Hover Popover 카드

- **트리거:** 참여자 ⓘ 아이콘 hover
- **딜레이:** 표시 0.3s / 숨김 0.2s
- **카드 내용:**
  ```
  ┌──────────────────────────┐
  │  박진우                   │
  │  한빛테크 · 대리          │
  │  ─────────────────────   │
  │  📧 jinwoo@hanbit.co.kr  │
  │  📱 010-1111-2222         │
  │  ─────────────────────   │
  │  수료 2/3 과정            │
  │  [상세보기 →]             │
  └──────────────────────────┘
  ```
- **표시 정보:**
  - 이름 (16px, bold)
  - 기업명 · 직위 (14px, 회색)
  - 구분선
  - 이메일
  - 연락처
  - 구분선
  - 수료 현황 (N/M 과정)
  - `[상세보기 →]` 링크 클릭 → `/participants` 페이지의 해당 참여자 드로어로 이동
- **위치:** 이름 기준 우측 또는 하단에 자동 배치 (화면 밖으로 나가지 않게)
- **카드 자체에 마우스 올리면** 카드 유지 (호버 영역 확장)

#### 5-6. 드로어 하단

- **고정 영역 아님** — 스크롤 가능
- 과정이 많을 경우 드로어 내부 스크롤
- 기업 정보 섹션은 상단 고정 고려 (sticky)

### 8. 수요조사 엑셀 업로드 모달

- **트리거:** 페이지 상단 `[업로드]` 버튼
- **형태:** 센터 모달
- **Step 1 — 파일 업로드:**
  - 드래그 앤 드롭 영역 + 클릭 업로드
  - 허용 파일: `xls`, `.xlsx`, `.csv`
  - 파일 선택 후 → 자동 파싱 → Step 2로 전환
- **Step 2 — 컬럼 매핑 확인:**
  - 파싱된 데이터 미리보기 테이블 (최대 5행)
  - 각 컬럼 헤더에 드롭다운: 시스템 필드와 매핑
    - 예: "회사명" 컬럼 → `기업명` 매핑
  - 매핑 안 되는 컬럼은 "건너뛰기" 선택 가능
- **Step 3 — 확인 및 등록:**
  - 총 N개 기업 등록 예정
  - 중복 기업 감지 시 경고 표시
  - `[등록]` → 일괄 등록 + 결과 요약 토스트

### 9. 상태 관리 및 데이터 흐름

- **기업 테이블 데이터:**
  - 전역 상태 관리 (Zustand / Context)
  - 필터 탭 변경 → 서버 재조회 or 클라이언트 필터링
  - 검색 → 클라이언트 필터링 (데이터 적을 시) or 서버 검색
- **드로어 데이터:**
  - 행 클릭 시 `companyId`로 상세 조회 API 호출
  - 교육 과정 + 세부 과정 + 참여자 = 한 번에 fetch (nested response)
  - 드로어 내 편집/추가 시 → API 호출 → 성공 시 로컬 상태 업데이트 + 테이블 반영
- **참여자 Popover:**
  - 참여자 ID 기반 조회 (캐싱 권장, 같은 사람 반복 hover 대비)
  - 또는 드로어 데이터에 참여자 기본 정보 포함하여 추가 API 불필요하게 설계

### 10. 반응형 및 예외 처리

- **반응형:**
  - 1280px 이상: 테이블 전체 컬럼 표시 + 드로어 480px
  - 1024px ~ 1279px: 이메일 컬럼 truncate + 드로어 400px
  - 1024px 미만: 드로어가 전체 화면 오버레이
- **빈 상태:**
  - 기업 0개 시: 일러스트 + "등록된 기업이 없습니다" + `[+ 기업 추가]` CTA
  - 검색 결과 0개 시: "검색 결과가 없습니다" 메시지
  - 드로어 내 교육 과정 0개 시: "등록된 교육 과정이 없습니다" + `[+ 과정 추가]` CTA
- **에러 처리:**
  - API 실패 시: 토스트 에러 메시지
  - 네트워크 오류 시: 재시도 버튼
- **삭제 확인:**
  - 기업 삭제, 과정 삭제, 참여자 제거 시 → 확인 다이얼로그
  - "삭제하면 관련 참여 기록도 함께 삭제됩니다" 경고

### 11. 하단 플로팅 액션바

- **트리거:** 체크박스 1개 이상 선택 시, 테이블 하단에서 - **slide-up 애니메이션**으로 등장

- **스타일 스펙:**

| 속성            | 값                                                   |
| --------------- | ---------------------------------------------------- |
| 위치            | `position: sticky; bottom: 0` (테이블 컨테이너 기준) |
| 높이            | 56px                                                 |
| 배경            | `white`, `border-top: 1px solid gray-200`            |
| 그림자          | `shadow-lg` (위쪽 방향, 떠 있는 느낌)                |
| 진입 애니메이션 | `translateY(100%) → translateY(0)`, 200ms ease-out   |
| 퇴장 애니메이션 | `translateY(0) → translateY(100%)`, 150ms ease-in    |
| 좌측 텍스트     | `font-weight: 600`, 선택 수 실시간 반영              |
| z-index         | 10 (드로어보다 낮게)                                 |

#### 11-1. 플로팅 버튼 동작 정의

#### 이메일 발송 (Primary)

- **스타일:** `bg-green-600 text-white rounded-md px-4 py-2`
- **동작:** 클릭 시 이메일 발송 모달 오픈
- **모달 내용:**
  - 수신자 목록: 선택된 기업의 **담당자 이메일** 자동 채움
  - 제목 / 본문 입력 (템플릿 선택 가능)
  - 발송 대상 미리보기: `한빛테크(담당자: 홍길동), 미래정밀(담당자: 박지은)...`
- **담당자 이메일 없는 기업:** ⚠️ 경고 표시 ("이메일 미등록 기업 N개 제외됨")

#### 📥 내보내기 (Secondary)

- **스타일:** `bg-gray-100 text-gray-700 rounded-md px-4 py-2`
- **동작:** 선택된 기업만 필터링하여 엑셀 다운로드
- **내보내기 포함 정보:**
  - 기업 기본 정보 (기업명, 소재지, 대표자, 담당자, 연락처, 이메일)
  - 협약 상태 + 협약 체결일
  - 참여 과정 목록
  - 소속 참여자 리스트 (하위 행으로 펼침)

#### ✕ 선택 해제 (Ghost)

- **스타일:** `bg-gray-100 text-gray-500 rounded-md px-4 py-2`
- **동작:** 모든 체크박스 해제 → 플로팅 바 slide-down 퇴장
- **키보드:** `Esc` 키로도 선택 해제 가능

#### 11-2. 체크박스 인터랙션 규칙

| 동작                      | 결과                                        |
| ------------------------- | ------------------------------------------- |
| 헤더 ☐ 클릭               | 현재 페이지 전체 행 선택/해제 토글          |
